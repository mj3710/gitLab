## ● 표도 만들어요
- 파이프 | 와 하이픈 - 으로 행과 열을 구별해요
- 양쪽 끝 파이프는 생략 가능
- 헤더 셀 구분 시, 3개 이상의 하이픈이 필요해요
- Typora에선 ctrl + T로 쉬운 생성!
- 행 늘릴 땐 ctrl + Enter
- :가 오른쪽에만 있으면 오른쪽 정렬
- :가 왼쪽에만 있으면 왼쪽 정렬
- :가 양쪽이면 가운데 정렬
<br>
<br>

| 명령어 |              설명               |
| :----: | :-----------------------------: |
| mkdir  |            폴더 생성            |
| touch  |            파일 생성            |
|   ls   |   현재 폴더의 파일 목록 출력    |
|   cd   |        다른 폴더로 이동         |
|   rm   | 파일 삭제 / 폴더 삭제 (-r 옵션) |
